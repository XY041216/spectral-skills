# Agent And Tool Division

Spectral Skills follow a knowledge-flow-first architecture.

## Agent Responsibilities

The Agent:

- understands user intent;
- reads rules and relevant references;
- selects relevant knowledge fragments;
- proposes a plan;
- asks required confirmations;
- chooses which MCP tool or script to call;
- interprets tool outputs within Contract rules.

## Tool And Script Responsibilities

Tools and scripts:

- preview files;
- execute confirmed plans;
- perform numerical calculations;
- validate schemas;
- write packages;
- compute metrics;
- train models;
- generate figures and reports.

## No Fabricated Artifacts

The Agent must not invent X, y, sample IDs, band axes, split indices, model
metrics, selected features, package files, or report artifacts. Numeric and file
outputs must be generated by tools or scripts and recorded in Contracts.

The Agent may summarize, explain, and route results, but computed artifacts
belong to deterministic execution.
