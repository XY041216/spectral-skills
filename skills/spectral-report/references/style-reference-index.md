# Style Reference Index

Establish the figure contract first. Inspect only the minimum relevant assets under `assets/style-references/`.

| Figure request | Inspect | Learn | Do not copy |
|---|---|---|---|
| Raw/transformed/class-mean spectra | `spectra-multipanel.png`, `palette-reference.svg` | panel alignment, axes, curve hierarchy | rainbow curves, heavy type |
| Repeated model comparison | `model-boxplot-grid.png`, `palette-reference.svg` | grouping, raw-point overlay, shared grammar | vague `Mean (%)`, boxes without repetitions |
| Hyperparameter response | `hyperparameter-line-grid.png`, `training-ratio-line-comparison.png` | ordered x, fixed model colors, small multiples | nominal model lines, crowded value labels |
| Embedding/latent space | one of `embedding-scatter-dense.png`, `embedding-scatter-separated.png` | class mapping, marker scale, spacing | treating separation as performance proof |

| Soft trend grid | `nature-pastel-trend-grid.jpg` | low-saturation families, shared scales | pixel-level replication |

Never paste, crop, trace, digitize, or sample scientific values from these files. Use explicit palette hex values from `references/visual-system.md`; derive layout from the current claim and data.

The three `nature-*.jpg` files are internal visual references derived from published material. Keep them in private/local distributions unless redistribution rights are confirmed. For a public marketplace release, omit them if rights are uncertain and retain the palette card plus written rules.
